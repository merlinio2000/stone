

# UART
[[UART]]

serial ports e.g. RS232

- TX/RX + optional control signal
- ONLY point-to-point
- full-duplex
- support for parry bit


# SPI
[[SPI - Serial Peripheral Interface]]

4 wire bus

- MOSI,MISO,SCLK,SS
- point to multipoint
- full duplex
- no built-in error detection


# I2C
[[I2C]]

2 wire bus

- SCL, SDA
- (multi)  point-to-multi-point
- half-duplex
- 7/10 bit slave address
- no built-in error detection
